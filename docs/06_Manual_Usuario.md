# 06. Manual de Usuario

## 1. Ingreso al sistema

1. Abra la aplicación en el navegador.
2. En la pantalla de inicio verá el formulario de login.
3. Si es administrador, use las credenciales:
   - usuario: admin
   - contraseña: admin123
4. Si es cliente, puede crear una cuenta nueva desde el botón “Crear cuenta”.

## 2. Uso como cliente

Una vez autenticado como cliente, podrá:
- ver el catálogo de libros
- consultar precios y stock
- navegar sin acceder a funciones administrativas

## 3. Uso como administrador

Una vez autenticado como administrador, podrá:
- gestionar libros
- registrar nuevas ventas
- consultar historial de ventas
- ver reportes básicos de ventas

## 4. Gestión de libros

### Registrar un libro
1. Ingresar al módulo de libros.
2. Completar el formulario con ISBN, título, autor, precio, stock y categoría.
3. Guardar.

### Editar un libro
1. Buscar el libro en la tabla.
2. Seleccionar la opción Editar.
3. Modificar los campos deseados.
4. Guardar cambios.

### Eliminar un libro
1. Seleccionar la opción Eliminar sobre el libro correspondiente.
2. Confirmar la acción.

## 5. Registro de ventas

1. Ingresar al módulo de ventas.
2. Seleccionar el libro o libros a vender.
3. Indicar la cantidad deseada.
4. Confirmar la venta.
5. El sistema descontará el stock automáticamente.

## 6. Cierre de sesión

Para cerrar sesión, use el botón correspondiente en la interfaz.
